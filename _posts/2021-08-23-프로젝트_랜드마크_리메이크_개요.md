# 1. 프로젝트 개요
* 내가 수강했던 국비교육 과정중 솔트룩스에서 진행했던  
 '최신인공지능 기반의 SW전문가 양성과정'  
 이라는 과정이 있었는데, 이때 마무리를 겸한 팀 프로젝트로서  
 '챗봇을 활용한 랜드마크 위치 서비스'  
 를 제작했었다.
* 해당 프로젝트를 간략히 설명하자면 이런 느낌이었다.
  1. 사용자가 랜드마크 이미지를 업로드한다
  2. 랜드마크 이미지를 학습한 모델이 해당 랜드마크가 어느것인지 예측한다.
  3. 해당 랜드마크의 위치정보를 알려준다.
  4. 사용자의 추가 요청시, 인근의 관광지/식당 등의 정보를 추가적으로 제공한다.
* 자세한 정보는 [교육 당시 발표자료] (/assets/files/솔트룩스 이미지 인식 프로젝트.pptx "ppt 다운로드 링크로 이어집니다") 를 통해 확인 가능하다.
